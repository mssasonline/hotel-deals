# Hotel Deals

A hotel booking platform built with Next.js, Supabase, and Tailwind CSS. Allows users to search hotels, view live deals, and complete bookings with payment.

## Features

- Hotel search with autocomplete
- Last-minute deals engine
- Room availability calendar
- Booking flow with payment
- Partner and admin dashboards
- Multi-language support (Arabic / English)

## Tech Stack

- **Frontend:** Next.js 14, Tailwind CSS
- **Backend:** Supabase (PostgreSQL + Auth + RLS)
- **State:** Zustand
- **Email:** Resend

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

Copy `.env.local.example` to `.env.local` and fill in your Supabase and other credentials.
